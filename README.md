# Projet Shooter

by fFukurou
### Third 3D game I make while learning the ropes of game development on Godot.

------------

### The prototype is a 1st person shooter in an alien setting.

- W, A, S, D --> Move
- Left Click --> Shoot
- Hold Right Click --> Zoom
- 1, 2, Mouse Scroll Wheel --> Change Weapon
